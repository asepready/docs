# Panduan Konfigurasi

## Ringkasan

Panduan konfigurasi sistem BSD: jaringan, layanan, storage, dan keamanan.

## Topik

1. `/etc/rc.conf` dan layanan
2. Konfigurasi jaringan (interface, routing, DNS)
3. Konfigurasi PF firewall
4. ZFS dan filesystem
5. Jails dan layanan aplikasi

## File Konfigurasi Penting

- `rc.conf` - layanan dan variabel sistem
- `pf.conf` - firewall
- `resolv.conf` - DNS

Lihat juga folder **configs/** untuk contoh konfigurasi.

# Glosarium Istilah Teknis BSD

## Ringkasan

Daftar istilah teknis yang sering digunakan dalam materi BSD Engineer Training.

## Istilah

- **BSD** - Berkeley Software Distribution
- **pkg** - Binary package manager (FreeBSD)
- **Ports** - Ports Collection (build from source)
- **PF** - Packet Filter (firewall)
- **ZFS** - Zettabyte File System
- **UFS** - Unix File System
- **rc.conf** - Konfigurasi layanan dan variabel sistem
- **Jail** - Lightweight virtualization/container (FreeBSD)

*(Daftar dapat diperpanjang per modul.)*

# Panduan Instalasi BSD

## Ringkasan

Panduan ini menjelaskan langkah instalasi FreeBSD, OpenBSD, dan NetBSD.

## Topik

1. Persiapan (hardware, VM, media)
2. Proses instalasi per varian
3. Post-installation (pkg, user, network)
4. Verifikasi instalasi

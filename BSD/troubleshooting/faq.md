# FAQ - Troubleshooting BSD

## Umum
- **Q: Sistem tidak boot?** A: Coba single-user mode, cek fstab dan disk.
- **Q: Tidak bisa SSH?** A: Cek sshd_enable, pf rules, network.
- **Q: Package install gagal?** A: pkg update, cek DNS dan network, repositori.

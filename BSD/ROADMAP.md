# Learning Roadmap — BSD Engineer Training

## Timeline Overview

```
Week 1-2:    Module 01 - Introduction (install, variants, post-install)
Week 3:      Module 02 - Package Management (pkg, ports)
Week 4:      Module 03 - User Management
Week 5-6:    Module 04 - Storage & Filesystem (disk, ZFS)
Week 7:      Module 05 - Network Configuration
Week 8:      Module 06 - PF Firewall
Week 9:      Module 07 - System Services (rc.conf, rc.d, cron, periodic)
Week 10:     Module 08 - Monitoring & Logging
Week 11:     Module 09 - Security Hardening
Week 12:     Module 10 - Backup & Recovery
Week 13:     Module 11 - Shell Scripting
Week 14-16:  Module 12 - Case Studies & Projects (web, firewall, mail, NAS, HA)
```

## Checkpoint System

| Module | Checkpoint | Validation |
|--------|------------|------------|
| 01 | Installation | FreeBSD/NetBSD/OpenBSD installed, lab-01 done |
| 02 | Packages | pkg/ports used, lab-02 done |
| 03 | Users/Groups | lab-03 done, permissions understood |
| 04 | Storage | ZFS or UFS working, lab-04 done |
| 05 | Network | Interfaces, VLAN, DNS, lab-05 done |
| 06 | Firewall | PF rules, NAT, lab-06 done |
| 07 | Services | rc.conf, rc.d, cron, lab-07 done |
| 08 | Monitoring | Syslog, rotation, lab-08 done |
| 09 | Security | Hardening, securelevel, quiz-09 |
| 10 | Backup | Backup strategy, lab-10 done |
| 11 | Scripting | Scripts functional, lab-11 done |
| 12 | Case Studies | At least one project (01–05) complete |

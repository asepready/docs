# Konfigurasi VMware untuk Lab BSD

## Ringkasan

- New Virtual Machine → Other → FreeBSD 64-bit
- RAM 1024 MB+, 2 vCPU
- Disk 20 GB+
- Network: NAT atau Bridged
- Mount ISO FreeBSD untuk instalasi

## Catatan

Pastikan VMware Tools (open-vm-tools) diinstal setelah FreeBSD berjalan untuk fitur tambahan.

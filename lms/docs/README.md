## LMS Laravel 12 – Dokumentasi

Repo ini berisi aplikasi LMS berbasis **Laravel 12** yang:

- Mendukung skala hingga **120.000 siswa** dan **200 guru/admin**.
- Menggunakan model database LMS umum (terinspirasi dari sumber seperti [`databasesample.com`](https://databasesample.com/database/lms)).
- Terintegrasi dengan materi `Linux` dan `BSD` di dalam repo dokumentasi utama melalui referensi path modul.

### Struktur dokumen di folder `docs`

- `architecture.md` – ringkasan arsitektur aplikasi, stack teknologi, dan deployment.
- `domain-model.md` – desain entitas database dan relasi utama LMS.
- `dev-process-agile-scrum.md` – panduan proses pengembangan menggunakan pendekatan Agile/Scrum.

Catatan tambahan ada di `README-LMS.md` di root proyek `lms`.

# Langkah Instalasi

## Ringkasan

Langkah instalasi Ubuntu Server / Rocky Linux: media (ISO/USB), partisi (LVM atau manual), konfigurasi dasar (user, SSH, network).

## Topik

- Persiapan media (Rufus, Ventoy, atau PXE)
- Proses installer (Ubuntu subiquity vs Anaconda)
- Disk partitioning dan LVM
- Pemilihan paket (minimal server)

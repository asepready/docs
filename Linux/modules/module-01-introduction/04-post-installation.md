# Post-Installation

## Ringkasan

Langkah setelah instalasi: apt update/upgrade atau dnf update, konfigurasi jaringan, user dan SSH, verifikasi systemd.

## Topik

- Update sistem (apt/dnf)
- Konfigurasi jaringan (Netplan atau nmcli)
- User, sudo, dan sshd
- Verifikasi: systemctl, journalctl, ip, ss

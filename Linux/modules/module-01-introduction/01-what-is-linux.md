# Apa itu Linux

## Ringkasan

Linux adalah kernel open-source yang dikembangkan oleh Linus Torvalds. Bersama GNU tools dan komponen lain membentuk sistem operasi lengkap (GNU/Linux).

## Topik

- Sejarah kernel Linux
- Filosofi open source dan lisensi GPL
- Perbedaan kernel Linux vs BSD
- Peran distribusi (packaging, konfigurasi default)

# Panduan Keamanan & Hardening

## Ringkasan

Panduan mengamankan dan mengeraskan sistem Linux untuk lingkungan produksi.

## Topik

1. Update dan patch (apt/dnf, unattended-upgrades)
2. SSH hardening (key-only, fail2ban)
3. Firewall dan filtering (ufw/iptables/nftables)
4. SELinux/AppArmor (mandatory access control)
5. Audit dan monitoring (auditd, log)
6. User, permission, dan sudo

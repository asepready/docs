# Panduan Troubleshooting

## Ringkasan

Panduan mengidentifikasi dan menyelesaikan masalah umum pada sistem Linux.

## Topik

1. Boot dan systemd (journalctl, rescue mode)
2. Jaringan dan konektivitas (ip, ss, nmcli)
3. Disk, LVM, dan filesystem
4. Layanan yang gagal (systemctl status, log)
5. Permission dan akses
6. Insiden keamanan

## Lebih Lanjut

Lihat folder **troubleshooting/** untuk panduan detail per kategori dan prosedur recovery.

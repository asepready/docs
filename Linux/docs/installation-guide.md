# Panduan Instalasi Linux

## Ringkasan

Panduan ini menjelaskan langkah instalasi distribusi Linux utama: Ubuntu Server, Debian, dan RHEL/CentOS/Rocky.

## Topik

1. Persiapan (media, VM, network)
2. Proses instalasi per distribusi (Ubuntu/Debian vs RHEL-based)
3. Post-installation (update, user, SSH, network)
4. Verifikasi instalasi (systemd, package manager)

# Best Practices

## Ringkasan

Praktik terbaik dalam administrasi dan operasi sistem Linux.

## Topik

1. Manajemen paket (apt/dnf, pinning, repository)
2. Backup dan disaster recovery (rsync, LVM snapshot, restic)
3. Monitoring dan logging (journald, rsyslog, Prometheus)
4. Dokumentasi konfigurasi (Infrastructure as Code)
5. Keamanan operasional (least privilege, audit)
6. Update dan maintenance terjadwal

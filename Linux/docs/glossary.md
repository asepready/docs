# Glosarium Istilah Teknis Linux

## Ringkasan

Daftar istilah teknis yang sering digunakan dalam materi Linux Engineer Training.

## Istilah

- **systemd** - Init system dan service manager (PID 1)
- **apt / dnf** - Package manager (Debian/Ubuntu vs RHEL/CentOS)
- **iptables / nftables** - Firewall dan packet filtering
- **firewalld** - Dynamic firewall daemon (RHEL/Fedora)
- **LVM** - Logical Volume Manager
- **ext4 / xfs** - Filesystem Linux
- **Netplan / NetworkManager** - Konfigurasi jaringan
- **SELinux / AppArmor** - Mandatory Access Control (MAC)
- **journald** - Logging systemd
- **cron / systemd timer** - Penjadwalan tugas

*(Daftar dapat diperpanjang per modul.)*

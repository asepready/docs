# Konfigurasi VMware untuk Lab Linux

## Ringkasan

- New Virtual Machine → Linux → Ubuntu 64-bit atau Other Linux 3.x 64-bit
- RAM 1024 MB+, 2 vCPU
- Disk 20 GB+
- Network: NAT atau Bridged
- Mount ISO untuk instalasi

## Catatan

open-vm-tools diinstal setelah OS berjalan untuk fitur integrasi VMware.

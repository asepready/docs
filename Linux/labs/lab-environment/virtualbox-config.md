# Konfigurasi VirtualBox untuk Lab Linux

## Ringkasan

- Buat VM baru, tipe Linux 2.6/3.x/4.x 64-bit (Ubuntu atau Generic)
- RAM minimal 1024 MB, 2 CPU disarankan
- Network: NAT atau Bridged
- Storage: VDI dinamis, minimal 20 GB
- Mount ISO Ubuntu Server atau Rocky Linux untuk instalasi pertama

## Langkah

1. New VM → Linux 64-bit (Ubuntu atau Other)
2. Memory: 1024 MB atau lebih
3. Create virtual disk → VDI, dynamically allocated, 20 GB+
4. Settings → Storage → pilih ISO
5. Start dan ikuti installer

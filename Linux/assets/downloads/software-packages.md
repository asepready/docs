# Software Packages (Linux)

Daftar paket umum untuk pelatihan:

- **Web:** nginx, apache2, php-fpm
- **Database:** mysql-server, postgresql
- **Tools:** vim, curl, wget, htop, tmux
- **Network:** net-tools, iproute2, bind9 (optional)
- **Samba/NFS:** samba, nfs-kernel-server

Instal via `apt install` (Ubuntu/Debian) atau `dnf install` (Rocky/RHEL).

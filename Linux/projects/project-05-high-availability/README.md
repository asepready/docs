# Proyek 5: High Availability (Linux)

## Deskripsi
Skenario HA: keepalived, HAProxy, atau redundant services dengan failover. Optional: Pacemaker/Corosync.

## Requirements
- Minimal 2 node atau VIP (keepalived)
- Shared storage atau replication (opsional)
- Failover procedure

## Deliverables
- Arsitektur, requirements, implementation, testing, documentation.

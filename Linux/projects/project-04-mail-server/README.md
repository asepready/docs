# Proyek 4: Mail Server (Linux)

## Deskripsi
Server mail (Postfix, Dovecot) dengan anti-spam, TLS, optional webmail.

## Requirements
- Ubuntu/Rocky, Postfix, Dovecot
- PF/SPF/DKIM (opsional)
- ufw/firewalld (25, 587, 993, 80/443)

## Deliverables
- Arsitektur, requirements, implementation, testing, documentation.

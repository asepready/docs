# Proyek 2: Firewall Gateway (Linux)

## Deskripsi
Gateway dengan iptables/ufw/firewalld: NAT, filtering, routing untuk jaringan internal.

## Requirements
- Ubuntu/Rocky, minimal 2 interface (WAN, LAN)
- ip_forward, NAT (masquerade)
- ufw atau firewalld atau iptables

## Deliverables
- Arsitektur, requirements, implementation, testing, documentation.

# Panduan Kontribusi

Terima kasih atas minat Anda untuk berkontribusi pada Linux Engineer Training.

## Cara Berkontribusi

1. **Fork** repositori ini
2. Buat **branch** untuk fitur atau perbaikan (`git checkout -b fitur/nama-fitur`)
3. **Commit** perubahan dengan pesan yang jelas
4. **Push** ke branch dan buat **Pull Request**

## Standar

- Gunakan Markdown untuk dokumentasi
- Script shell: gunakan `#!/bin/bash` atau `#!/bin/sh`, portable bila memungkinkan
- Konfigurasi: sertakan komentar untuk opsi penting
- Bedakan contoh untuk Debian/Ubuntu (apt) vs RHEL/CentOS (dnf/yum) bila perlu

## Pertanyaan?

Buka Issue untuk diskusi atau pertanyaan.

# Script Git

## Merge ke main

Menggabungkan branch saat ini ke `main` lalu push.

**Windows (PowerShell):**
```powershell
cd c:\docs
.\Linux\scripts\git\merge-to-main.ps1
```

**Linux / Git Bash:**
```bash
cd /path/to/repo
chmod +x Linux/scripts/git/merge-to-main.sh
./Linux/scripts/git/merge-to-main.sh
```

Pastikan perubahan sudah di-commit di branch saat ini sebelum menjalankan.

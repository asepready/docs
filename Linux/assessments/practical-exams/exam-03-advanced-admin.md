# Ujian Praktik 03: Advanced Administration

## Waktu
120 menit

## Tujuan

1. Buat LV, snapshot, backup
2. Konfigurasi logrotate dan verifikasi log
3. Buat systemd service dan timer
4. Prosedur restore dari backup
